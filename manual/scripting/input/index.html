概述

目前 Fireball 使用类 web 开发的注册机制来响应玩家输入，分为全局和 Entity 两种注册途径。能够获得的输入有键盘、鼠标和触摸操作，触摸操作实现了模拟鼠标(Mouse Simulation)的效果，只要注册鼠标事件就能接收到触摸操作。

反注册时传入的事件类型和回调函数必须和注册时一致。

Entity注册
